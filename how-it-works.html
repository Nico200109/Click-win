Comment ça marche

  1. Le prix d’un objet baisse automatiquement.
  2. Vous pouvez bloquer le prix quand il vous convient.
  3. Vous avez un temps limité pour payer (10 secondes par défaut).
  4. Si le paiement n’est pas effectué, l’objet repart en enchère.
  5. Les enchères tournent automatiquement entre plusieurs objets.
