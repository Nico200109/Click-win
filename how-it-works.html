Comment ça marche

1. Choisir une enchère
Sélectionnez l'objet que vous voulez tenter de gagner.
2. Surveiller le prix
Le prix baisse automatiquement jusqu'à un plancher minimum.
3. Bloquer le prix
Appuyez sur le bouton « Bloquer le prix » pour sécuriser l'enchère.
4. Payer et gagner
Effectuez le paiement dans le temps imparti et votre nom apparaît sur la page Gagnants.
